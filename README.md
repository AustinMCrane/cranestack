# CraneStack

A monorepo boilerplate for a three-component stack:

| Component | Tech |
|---|---|
| REST API | Go, chi, SQLite |
| MCP Server | Go, mark3labs/mcp-go, HTTP/SSE + stdio |
| iOS App | SwiftUI, iOS 18, Apple Sign-In |

## Architecture

```
iOS App  ──── POST /auth/login ──────────────► Go REST API (port 8080)
         ◄─── session token ─────────────────
         ──── POST /auth/generate-mcp-key ──►
         ◄─── PAT ──────────────────────────

AI Agent ──── MCP tool calls ───────────────► Go MCP Server (port 9090)
              (via PAT)                        │
                                              └─► Go REST API
```

## Prerequisites

- Go 1.23+
- Xcode 16+ (for iOS)
- [xcodegen](https://github.com/yonaskolb/XcodeGen) (`brew install xcodegen`)

## Running

### Go REST API

```bash
# Development (with a dev session token for testing without Apple Sign-In)
make run-api

# Production
go run ./cmd/api
```

### Go MCP Server

```bash
# HTTP/SSE transport (default, port 9090)
make run-mcp PAT=<your-pat>

# stdio transport (for local AI agent use)
MCP_PAT_TOKEN=<your-pat> MCP_TRANSPORT=stdio go run ./cmd/mcp
```

### iOS App

1. Open `ios/CraneStack.xcodeproj` in Xcode
2. Select a simulator or device
3. Build & Run (`⌘R`)
4. In development, tap **Dev Login** to bypass Apple Sign-In

To regenerate the Xcode project after editing `ios/project.yml`:
```bash
cd ios && xcodegen generate
```

## Environment Variables

### API Server (`cmd/api`)

| Variable | Default | Description |
|---|---|---|
| `DB_PATH` | `cranestack.db` | Path to the SQLite database file |
| `PORT` | `8080` | Port the API listens on |
| `DEV_SESSION_TOKEN` | _(unset)_ | When set, registers a dev session for local testing without Apple Sign-In |
| `LOG_FORMAT` | `json` | Log format: `json` or `text` |

### MCP Server (`cmd/mcp`)

| Variable | Required | Description |
|---|---|---|
| `MCP_PAT_TOKEN` | ✅ | Personal Access Token generated by the API |
| `MCP_TRANSPORT` | `sse` | Transport to use: `sse` or `stdio` |
| `API_BASE_URL` | `http://localhost:8080` | Base URL of the REST API |
| `LOG_FORMAT` | `json` | Log format: `json` or `text` |

## PAT Flow (Development)

```bash
# 1. Start the API with a known dev session token
make run-api

# 2. Generate a PAT using that dev token
curl -s -X POST http://localhost:8080/auth/generate-mcp-key \
  -H "Authorization: Bearer dev-session-token" | jq

# 3. Start the MCP server with that PAT
make run-mcp PAT=<token from step 2>
```

```
│   │   ├── routes.go         # Route wiring
│   │   ├── server.go         # Server struct
│   │   └── session.go        # In-memory session store
│   ├── db/
│   │   ├── db.go             # SQLite open + migrate
│   │   ├── schema.go         # CREATE TABLE statements
│   │   └── repository.go     # Data access methods
│   └── mcp/
│       ├── client.go         # Authenticated HTTP client
│       ├── server.go         # MCP server + transport selection
│       └── tools/            # MCP tool definitions

    ├── project.yml           # xcodegen spec
    └── CraneStack/
        ├── CraneStackApp.swift
        ├── AuthManager.swift
        ├── APIClient.swift
        └── Views/
```
